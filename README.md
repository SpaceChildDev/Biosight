# Biosight

An iOS app to securely track lab results, medical documents, and Apple Health data in one place.

---

## Features

- **PDF Import** — Upload lab reports and automatically parse them with AI
- **Apple Health Sync** — Automatically sync heart rate, sleep, step count, and other metrics
- **Trend Charts** — Visualize how your values change over time
- **Multiple Profiles** — Track family members with separate profiles
- **Reference Comparison** — Instantly spot abnormal values
- **Easy Mode** — Large buttons and a simplified interface for older users
- **Privacy-Focused** — No server, all data stays on device and in iCloud

## Tech Stack

- Swift / SwiftUI
- SwiftData (local database)
- HealthKit
- Google Gemini API (optional — user provides their own key)
- iOS 17+

## Privacy

Your health data is stored only on your device and in your iCloud account. The app has no server of its own. AI analysis is entirely optional.
